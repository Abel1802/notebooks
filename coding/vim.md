## Insert mode

### delete

- **ctrl+h**: delete a character

- **ctrl+w**: delete a word

- **ctrl+u**: delete a line

---

## Normal mode

### replace

- **:%s/before/after/g**

---

## Visual mode

### select

- **v**: character-wise
- **V**: line-wise
- **ctrl+v**: block-wise
